使用Github和Hexo搭建一个自己的博客

1.注册一个自己的Github账号

  • https://github.com在该网站可完成注册，这里就不赘述

2.创建一个仓库

  • 点击右上角的new即可创建一个仓库，将仓库的Repository name明明为username.github.io

  • username是注册github的用户名。点击最下方的创建即可。

3.安装Hexo

备注：本人的是mac系统

  • 需要的依赖：git node.js nvm
  • 具体的安装方式-brew的安装比较复杂，这里不赘述
    • brew install git
    • brew install nvm
    • brew install node
  • hexo的安装方式
    • 首先需要换源，不然很难安装上npm config set registry https://registry.npm.taobao.org
    • sudo npm install hexo-cli -g

4. 发布博客

  • 创建一个文件夹：执行hexo init

  • 修改配置文件

    • _config.yml

      • deploy:

        type: git

        repo: https://github.com/username/username.github.io

        branch: master

    • 这是一般会出错，需要执行以下命令

      • sudo npm install –save hexo-deployer-git

  • 发布到github仓库

    • hexo g && hexo g

  • 查看

    • 输入网址https://username.github.io即可查看
